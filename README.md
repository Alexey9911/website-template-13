# Nexus Web Agency

![Website Template Banner](image.png)

A cutting-edge web design and development agency website featuring modern Web3 aesthetics, smooth animations, and interactive project showcases. Built with advanced technologies and optimized for exceptional user experience.

## 🎯 Features

- **Modern Web3 Design** - Futuristic interface with blockchain-inspired aesthetics
- **Advanced Animations** - Smooth GSAP animations with scroll triggers and split text effects
- **Interactive Project Cards** - Sticky cards with dynamic content presentation
- **Smooth Scrolling** - Lenis integration for buttery-smooth scroll experience
- **Loading Animations** - Professional loader with progress tracking
- **Responsive Navigation** - Animated hamburger menu with smooth transitions
- **Contact Forms** - Interactive contact form with modern styling
- **Performance Optimized** - Fast loading and optimized for all devices

## ⚡ Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/Alexey9911/website-template-13.git
   cd website-template-13
   ```

2. **Open in your browser**
   Simply open `index.html` in your preferred web browser or use a local development server.

3. **Customize**
   - Update content in HTML files
   - Modify styles in CSS files
   - Replace images in `/assets` folder
   - Adjust animations in `/js` files

## 🎨 Customization

The template is highly customizable:

- **Colors**: Update CSS custom properties in `global.css`
- **Content**: Edit HTML content directly
- **Images**: Replace files in `/assets` folder
- **Animations**: Modify GSAP timelines in JavaScript files
- **Layout**: Adjust CSS Grid and Flexbox properties
